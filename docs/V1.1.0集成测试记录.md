# V1.1.0 集成与实例测试记录

## 测试环境

- 开发分支：`v1.1.0`
- Docker 测试镜像：`linx-xiling-dev-test:latest`

- 所有实例测试命令均显式提供 `--format`
- 临时下载样本、临时仓库副本和输出目录均位于系统临时目录，测试结束后删除

## 备注

- 从 ISO 提取 DEB 样本时跳过了部分 Rock Ridge symlink 别名，最终成功提取真实 DEB 包；这是已知 ISO 读取边界，不影响完整扫描结果。
- repo 扫描会写回 `assist/originators.json`，因此真实 repo 测试在临时仓库副本中执行，避免污染工作区。
- GBT 漏洞查询依赖本地 Elasticsearch/OSV 权限配置。本次环境中 ES 可达但认证失败，工具输出中文告警并生成空漏洞列表，符合受控内生产的降级预期。
